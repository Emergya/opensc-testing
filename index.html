PKCS#11 simple applet

If you get 'access denied (java.lang.RuntimePermission accessClassInPackage.sun.security.pkcs11)' error code, you must edit your ${java.home}/**/java.policy

//OpenSC testing permission
	grant codeBase "**opensc-testing-core-0.1-SNAPSHOT.jar" {
		permission java.util.PropertyPermission "sun.security.pkcs11.*", "read";
	};

This applet access to your card, list your certs and sign a text string using OpenSC

Tested on Ubuntu 12.04 with OpenSC installed from source and using entersafe card
