PKCS#11 simple applet

This applet access to your card, list your certs and sign a text string using OpenSC

Tested on Ubuntu 12.04 with OpenSC installed from source and using entersafe card

Known issues

Access Denied

If you get 

access denied (java.lang.RuntimePermission accessClassInPackage.sun.security.pkcs11)
error code, you must edit your ${java.home}/**/java.policy and add:

	//OpenSC testing permission
	grant codeBase "file:/${your_project_dir}/opensc-testing/*" {
		permission java.security.AllPermission;
	};

More

If you have other problems with the applet:

  1. Clear the cache browser
  2. Test smartcard connection with opensc-tools
  3. Try again
